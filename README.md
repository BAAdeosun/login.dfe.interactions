# login.dfe.interactions
Providing user interactions as part of the **login.dfe** project.

## Getting Started

Install deps
```
npm i
```

Setup Keystore & devlopment ssl certs
```
npm run setup
```

Run
```
npm run dev
```

Visit
```
https://localhost:4431/
```
