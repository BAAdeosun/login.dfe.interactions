# login.dfe.interactions
Providing user interactions as part of the **login.dfe** project.

## Getting Started

Install deps
```
npm i
```

Run
```
npm run dev
```

Visit
```
https://localhost:4431/
```

In 'dev mode' you should see a page listing all the available interactions
